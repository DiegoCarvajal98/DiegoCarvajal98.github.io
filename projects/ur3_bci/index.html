UR3 BCI

Control of a Robot Manipulator with EEG Data for Object Manipulation Tasks

Throughout the development of this project, I used my programming skills with python and C++ as well as Matlab to program a UR3 robot manipulator to pick one of five possible objects according to the result from a machine learning model and Brain-Computer Interface data and place it at a determined location using ROS for the robot control and object detection.
